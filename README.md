# TNWallpaper
可以把任何窗口（除了UWP应用）嵌入到桌面实现动态壁纸的软件

## 功能

- 多显示器支持，按位置稳定排序
- 视频壁纸（ffplay 播放，崩溃自动重启）
- 嵌入任意窗口到桌面背景
- 系统托盘（最小化到托盘，右键菜单）
- 精简版 Windows 兼容

## 依赖

- ffplay.exe（放程序同目录或系统 PATH）
- Windows 7+



## 使用

1. 把 ffplay.exe 放到程序目录
2. 运行程序，选择视频或窗口
3. 选择目标显示器（支持多屏同时播放）
4. 点击播放

**窗口嵌入模式**：点击"选择窗口"后，鼠标悬停高亮目标窗口，左键点击嵌入，右键或 ESC 取消。


## 快捷键

- ESC：取消窗口选择

## 托盘菜单

- 显示：恢复主窗口
- 停止：停止播放/嵌入
- 退出：关闭程序

## 注意事项

- 视频模式需要 ffplay.exe（FFmpeg 组件）
- 窗口嵌入不支持"所有显示器"选项
- 程序关闭时会自动恢复被嵌入的窗口
